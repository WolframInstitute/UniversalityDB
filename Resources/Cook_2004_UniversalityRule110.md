# Universality in Elementary Cellular Automata

Matthew Cook, 2004. Complex Systems 15(1), 1-40.

Proves Rule 110 is computationally universal by simulating cyclic tag systems. The encoding uses glider collisions on a periodic background to represent tag system symbols. Each tag step is simulated by a sequence of glider interactions that propagate along the Rule 110 spacetime. Overhead: spatial sigma ~ 20 (glider width), temporal tau ~ O(n) (glider travel time).

This is the key edge connecting cellular automata to the tag system / TM hierarchy. The proof was famously difficult due to the complexity of classifying all glider interactions in Rule 110.
