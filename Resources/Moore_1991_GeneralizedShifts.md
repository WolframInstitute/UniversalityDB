# Generalized Shifts: Unpredictability and Undecidability in Dynamical Systems

Cristopher Moore, 1991. Nonlinearity 4, 199-230.

Introduces generalized shifts as a dynamical systems model equivalent to Turing machines. Theorem 7: any TM is conjugate to a generalized shift via state-into-tape encoding (sigma=1, tau=1). Theorem 8: reverse direction, any generalized shift can be simulated by a TM (sigma=1, tau=O(w) where w is alphabet size). This establishes a tight bidirectional connection between TMs and GS.

The state-into-tape encoding merges the TM head state into the tape cell at the head position, using an extended alphabet of size s*k + k where s = states, k = symbols.
