# P-completeness of Cellular Automaton Rule 110

Turlough Neary and Damien Woods, 2006. ICALP, LNCS 4051.

Provides an alternative polynomial-overhead universality proof for Rule 110 via the chain: generic TM -> clockwise TM -> cyclic tag -> Rule 110. Each step has polynomial overhead, giving an overall polynomial simulation. This is significant because Cook's original proof has exponential overhead in some steps.
