# Generative Language Modeling for Automated Theorem Proving

Stanislas Polu and Ilya Sutskever, 2020. arXiv:2009.03393.

Introduces GPT-f, a generative language model for automated theorem proving in Metamath. Demonstrates that large language models can generate valid proof steps and close open Metamath theorems. Relevant as background methodology for the LLM-assisted proof pipeline in this project.

Download: https://arxiv.org/abs/2009.03393
