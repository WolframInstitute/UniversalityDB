# Cook 2004 — Universality of Rule 110

Matthew Cook. *Universality in Elementary Cellular Automata.* Complex Systems 15(1), 1-40, 2004.

## Summary

Proves that elementary cellular automaton Rule 110 is computationally universal by showing it can simulate any cyclic tag system using glider collisions on a periodic background. Spatial overhead ~20, temporal overhead O(n).

## Key results

- Rule 110 simulates cyclic tag systems via engineered glider collisions.
- Gliders propagate on a periodic background; collisions implement CTS operations.
- This connects the ECA world to the tag system / TM hierarchy.
