# Polu & Sutskever 2020 — GPT-f for Theorem Proving

> Status: **draft**

Stanislas Polu and Ilya Sutskever. *Generative Language Modeling for Automated Theorem Proving.* arXiv:2009.03393, 2020.

## Summary

Introduces GPT-f, a generative language model for automated theorem proving in Metamath. Demonstrates that large language models can generate valid proof steps and close previously open Metamath theorems. First major result showing LLMs as practical proof assistants.

## Use in this project

Background methodology for the LLM-assisted proof pipeline. Validates the approach of using generative models to propose proof steps that are then verified by a formal system (Metamath there, Lean 4 here).
