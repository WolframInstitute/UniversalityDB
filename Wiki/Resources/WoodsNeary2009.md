# Woods & Neary 2009 — Complexity of Small UTMs

> Status: **draft**

Damien Woods and Turlough Neary. *The Complexity of Small Universal Turing Machines: A Survey.* Theoretical Computer Science 410(4-5), 443-450, 2009.

## Summary

Comprehensive survey cataloging known small universal Turing machines by (states, symbols) pairs. Provides detailed overhead analysis for each. Key reference for identifying which machines to add as vertices in the universality graph.

Download: https://doi.org/10.1016/j.tcs.2008.09.051

## Use in this project

Reference for populating the universality graph with small UTMs and understanding overhead tradeoffs.
